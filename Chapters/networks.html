Chapter 10: Networks & Internet

How your "Hello" travels the world in milliseconds! 🌍✈️

What is a Network?

A network is a group of computers connected together to share data and resources.

🤝 Simple example: Your phone sending a WhatsApp message to your friend across the country — that's a network in action!

Network Types by Size

🏠

PAN

Personal Area Network
Very small (Bluetooth, your devices)

🏢

LAN

Local Area Network
Your home/office WiFi

🏙️

MAN

Metropolitan Area Network
Covers a whole city

🌍

WAN

Wide Area Network
The Internet itself!

Network Topologies (Shapes)

Bus Topology

All computers on ONE main cable. Simple but if cable breaks, network down! 📏

Star Topology

All connect to central hub (like a WiFi router). Most common today! ⭐

How Data Travels the Internet

When you send a message, it's broken into small pieces called packets 📦, routed through many computers (routers), and reassembled at destination!

The 4 Layers of Internet (TCP/IP)

The Internet works in 4 layers. Each layer does one job:

4. Application Layer
HTTP, Email, WhatsApp — what YOU see 👤
3. Transport Layer
TCP/UDP — breaks into packets, reassembles 📦
2. Network Layer
IP — finds the route (like GPS) 🗺️
1. Link Layer
Ethernet/WiFi — physical transmission ⚡
🎯 Remember: Your message goes DOWN the layers at sender, and UP the layers at receiver!

Protocols

A protocol = set of rules for communication. Like traffic rules for data! 🚦

  • TCP/IP: The main Internet protocol suite
  • HTTP/HTTPS: Web browsing
  • Ethernet (CSMA/CD): Wired networks
  • WiFi (CSMA/CA): Wireless
  • SMTP: Email sending

Summary

✅ Network = connected computers sharing data

✅ Types: PAN, LAN, MAN, WAN (smallest → biggest)

✅ Data travels in packets through routers

✅ Internet has 4 layers: Application → Transport → Network → Link

✅ TCP/IP = the rules that make Internet work
