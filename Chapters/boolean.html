Chapter 4: Boolean Logic

The secret language of computer decisions! 🧠✨

What is Boolean Logic?

Named after mathematician George Boole, Boolean logic uses only TRUE (1) or FALSE (0).

Computers use logic gates (tiny electronic switches) to make decisions. Let's learn the 4 main gates! 🚪

🤔 Why important? Your computer makes billions of these tiny yes/no decisions every second to run apps, play games, show videos, and more!
AND

AND Gate

Rule: Output is 1 only when BOTH inputs are 1.

🏠 Real-life: Door with TWO locks 🔐🔐. Door opens ONLY if BOTH locks are open. If even one is closed → door stays shut!

Quick Summary

✅ AND: BOTH must be 1 (strict 🔒)

✅ OR: AT LEAST ONE must be 1 (friendly 😊)

✅ XOR: DIFFERENT inputs give 1 (exclusive 🎭)

✅ NOT: Flips input (rebel 😈)
