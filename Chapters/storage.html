Chapter 6: Storage Systems

How computers remember everything — from your photos to your favorite games! 💾

Two Types of Memory

Main Memory (RAM)

Short-term memory 🧠

  • ⚡ Very FAST
  • 💨 Data LOST when power off
  • 📏 Small (4GB - 32GB)
  • 💰 Expensive
  • Used for: currently running apps
Like your desk — quick access, but cleared at end of day.

Mass Storage

Long-term memory 📚

  • 🐢 Slower
  • 💾 Data SAVED even if power off
  • 📦 HUGE (500GB - several TB)
  • 💵 Cheap
  • Used for: files, photos, videos, apps
Like your bookshelf — stores everything forever.

Storage Devices

💽

Magnetic Disks

Hard Disk Drives (HDD)
Tracks, sectors, seek time, rotation delay. Old but cheap!

⚡

Solid State (SSD)

Flash memory — no moving parts. Super fast! ⚡ Modern laptops use these.

💿

Optical Disks

CD (700MB), DVD (4.7GB), Blu-ray (25GB+). Read by laser 🔦

🔌

USB/Flash Drives

Portable flash memory. Plug in, carry data anywhere!

How a Hard Disk Works

A hard disk has spinning platters (like small DVDs) and magnetic heads that read/write data.

🎯 Key terms:
• Track = circular path on disk
• Sector = small section of a track
• Seek Time = time to move head to right track
• Rotation Delay = time waiting for right sector to come under head

Summary

✅ RAM = fast, temporary memory

✅ Hard Disk/SSD = permanent storage

✅ ROM = cannot be changed (read-only)

✅ Storage types: Magnetic, Optical, Flash

✅ Modern PCs use SSDs — much faster than HDDs!
