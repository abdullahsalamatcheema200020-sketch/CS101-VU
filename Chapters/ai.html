Chapter 15: Artificial Intelligence

Machines that think, learn, and see! 🤖🧠

What is AI?

Artificial Intelligence (AI) = building machines that can do tasks which normally need human intelligence.

🤖 Examples around you:
• ChatGPT — answers any question 💬
• Google Maps — finds best route 🗺️
• Face unlock on phones 📱
• Netflix recommendations 🎬
• Voice assistants (Siri, Alexa) 🎤

The Turing Test

Proposed by Alan Turing in 1950 to test if a machine is "intelligent":

🧪 The test:
A human chats with something (unseen) — could be a computer or another human.
If the human CAN'T tell which is the computer → the AI passes! ✅

Modern AI like ChatGPT often passes this test! 🎉

Main Areas of AI

Computer Vision

Machines that "see"! Image analysis uses:

  • Edge enhancement — find shapes
  • Smoothing — reduce noise
  • Pattern recognition

NLP

Natural Language Processing

  • Syntactic — grammar
  • Semantic — meaning
  • Contextual — situation

Machine Learning

Machines learn from data — get better with experience! 📈

Neural Networks

Inspired by the human brain — millions of connected neurons!

Intelligent Agents

An intelligent agent is a program that:

  1. 👀 Perceives its environment (sensors)
  2. 🧠 Thinks about what to do
  3. 🤖 Acts on the environment (actuators)
🚗 Example: Self-driving car
Sees road (cameras) → Thinks about traffic → Turns wheel/brakes!
