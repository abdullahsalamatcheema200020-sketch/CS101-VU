Chapter 8: CPU & Machine Cycle

Meet the brain of your computer! See it work step-by-step! 🧠⚡

What is a CPU?

CPU = Central Processing Unit. It's the brain of your computer — it does ALL the thinking and calculating! 🧠

Right now, your CPU is doing billions of operations per second to show you this webpage!

Parts of a CPU

🔢

ALU

Arithmetic Logic Unit
Does math (+, -, ×, ÷) and logic (AND, OR, NOT) operations.

🎛️

Control Unit

Directs traffic — tells all parts what to do and when.

📋

Registers

Tiny super-fast memory slots inside CPU for temporary data.

Important Registers

  • Program Counter (PC): Address of next instruction
  • Instruction Register (IR): Holds the current instruction being executed

The Machine Cycle (Fetch-Decode-Execute)

The CPU repeats 3 steps over and over, billions of times per second!

1. FETCH

Get instruction from memory

2. DECODE

Understand what to do

3. EXECUTE

Do the task!

Simulated Memory

0x01: LOAD A = 5
0x02: LOAD B = 3
0x03: ADD C = A + B
0x04: PRINT C
0x05: HALT STOP

CPU Architectures

RISC

Reduced Instruction Set Computer

Fewer, simpler instructions → very fast. Used in phones (ARM chips).

CISC

Complex Instruction Set Computer

More complex instructions → handles complex tasks in 1 step. Used in PCs (Intel/AMD).

Summary

✅ CPU = brain of computer

✅ Parts: ALU + Control Unit + Registers

✅ Machine Cycle: Fetch → Decode → Execute

✅ PC tracks next instruction, IR holds current one

✅ Types: RISC (simple, fast) vs CISC (complex, powerful)
