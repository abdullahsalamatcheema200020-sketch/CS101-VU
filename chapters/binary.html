Chapter 3: Bits & Binary

How computers speak using only 0s and 1s! 🔢

What is a Bit?

A Bit (Binary digIT) is the smallest piece of information a computer uses. It has only 2 possible values:

🔦 Think of a light switch!
• ON = 1 (electricity flowing 💡)
• OFF = 0 (no electricity 🌑)
That's it! Everything in a computer is just millions of these switches!

Decimal to Binary Conversion

How do we turn a normal number like 13 into binary? Use the Division Method:

📝 Steps:
1. Divide the number by 2
2. Write down the remainder (0 or 1)
3. Use the quotient as new number
4. Repeat until quotient = 0
5. Read remainders from bottom to top!

Example: Convert 13 to Binary

1
13 ÷ 2 = 6 → Remainder: 1
2
6 ÷ 2 = 3 → Remainder: 0
3
3 ÷ 2 = 1 → Remainder: 1
4
1 ÷ 2 = 0 → Remainder: 1
📖 Read bottom → top: 1, 1, 0, 1 → 1101
So, 13 in binary = 1101 ✨
